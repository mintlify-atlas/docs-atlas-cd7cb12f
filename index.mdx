## Install in 30 seconds

```elixir mix.exs
def deps do
  [
    {:httpower, "~> 0.20.0"},
    {:finch, ">= 0.19.0"}   # Recommended adapter — or use :req or :tesla
  ]
end
```

```elixir lib/my_app/application.ex
def start(_type, _args) do
  HTTPower.Logger.attach()  # Enable PCI-compliant logging

  children = [
    {Finch, name: HTTPower.Finch}
  ]
  Supervisor.start_link(children, strategy: :one_for_one)
end
```

```elixir
# Make your first request
{:ok, response} = HTTPower.get("https://api.example.com/users")
IO.inspect(response.status)  # 200
IO.inspect(response.body)    # %{"users" => [...]}
```
